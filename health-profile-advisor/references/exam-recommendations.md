# 分年龄段体检项目推荐方案

> 基于《健康体检基本项目专家共识》及主流医学建议整理，供 Phase 4 生成体检建议时按需读取。根据用户画像中的年龄、性别、家族史和已有异常指标进行个性化推荐。

---

## 一、基础必查项目（所有成年人每年）

| 类别 | 项目 |
|------|------|
| 一般检查 | 身高、体重、BMI、血压、腰围 |
| 内科 | 心肺听诊、腹部触诊 |
| 外科 | 皮肤、淋巴结、甲状腺触诊、肛门指检 |
| 眼科 | 视力、眼压、眼底 |
| 耳鼻喉 | 常规检查 |
| 口腔 | 牙齿、牙龈、口腔黏膜 |
| 血常规 | 全套 |
| 尿常规 | 全套 |
| 便常规+隐血 | 每年 |
| 肝功能 | ALT、AST、TBIL、ALB、GGT |
| 肾功能 | Cr、BUN、UA |
| 血脂 | TC、TG、HDL-C、LDL-C |
| 血糖 | FPG，建议加 HbA1c |
| 心电图 | 十二导联 |
| 胸部影像 | X光或低剂量CT |

---

## 二、分年龄段加查项目

### 20-29岁
重点关注：传染性疾病筛查、心理健康、生殖健康

| 项目 | 原因 |
|------|------|
| 乙肝五项 | 确认免疫状态 |
| 丙肝抗体 | 如有高危行为 |
| HIV/梅毒 | 如有高危行为 |
| 甲状腺功能 | 年轻女性高发 |
| HPV+TCT | 女性（21岁以上，每3年） |
| 乳腺超声 | 女性（有家族史提前） |

### 30-39岁
重点关注：代谢综合征早期信号、甲状腺、生殖健康

| 项目 | 原因 |
|------|------|
| HbA1c | 糖尿病早期筛查 |
| 甲状腺功能+超声 | 30+甲状腺结节高发 |
| 乳腺超声/钼靶 | 女性（每1-2年） |
| HPV+TCT | 女性（每3年或每5年联合筛查） |
| 精液常规 | 男性备孕 |
| 幽门螺杆菌 | 胃癌风险筛查 |
| 骨密度 | 仅低风险人群不查；有危险因素时加查 |

### 40-49岁
重点关注：心血管、肿瘤筛查起步、代谢疾病

| 项目 | 原因 |
|------|------|
| 颈动脉超声 | 心血管风险评估 |
| 心脏超声 | 心功能评估 |
| 动态心电图 | 有心悸等症状时 |
| 胃镜 | 有消化道症状/家族史 |
| 肠镜 | 45岁起建议首次筛查 |
| 肿瘤标志物 | CEA、AFP、CA199等（组合筛查） |
| 低剂量肺部CT | 吸烟者/被动吸烟/职业暴露 |
| 乳腺钼靶+超声 | 女性（每年） |
| PSA | 男性（有家族史从45岁起） |
| 骨密度 | 绝经后女性 |
| 眼底照相 | 糖尿病/高血压患者 |

### 50-59岁
重点关注：肿瘤全面筛查、心脑血管、骨密度

| 项目 | 原因 |
|------|------|
| 肠镜 | 每5-10年（或便潜血每年） |
| 胃镜 | 每2-3年 |
| 低剂量肺部CT | 每年 |
| 冠脉CT | 高危人群 |
| 颅脑MRI/MRA | 有症状/高危人群 |
| 主动脉超声 | 腹主动脉瘤筛查 |
| 骨密度 | 女性（每2年）；男性（每3年） |
| 维生素D | 骨质疏松风险评估 |
| 同型半胱氨酸 | 心脑血管风险评估 |
| 糖尿病全套 | HbA1c+空腹胰岛素+餐后血糖 |

### 60岁以上
重点关注：慢病管理、功能评估、跌倒风险

| 项目 | 原因 |
|------|------|
| 以上所有项目按需加强 | — |
| 认知功能评估 | 阿尔茨海默病早期筛查 |
| 跌倒风险评估 | 肌力、平衡 |
| 营养评估 | 白蛋白、前白蛋白、维生素 |
| 肺功能 | 慢阻肺筛查 |
| 前列腺超声 | 男性 |
| 全腹超声 | 每1-2年 |

---

## 三、家族史加查项目

| 家族病史 | 建议加查项目 | 开始筛查年龄 |
|---------|-------------|-------------|
| 直系亲属糖尿病 | HbA1c、空腹胰岛素、OGTT | 比亲属发病年龄早10年 |
| 直系亲属心血管病 | 颈动脉超声、同型半胱氨酸、脂蛋白a | 40岁起或提前 |
| 直系亲属肿瘤（任一类型） | 对应肿瘤标志物+影像筛查 | 比亲属发病年龄早10-15年 |
| 家族性结直肠癌 | 肠镜 | 40岁或比最早发病者早10年 |
| 家族性乳腺癌 | 乳腺MRI+钼靶+超声 | 30岁起 |
| 高血压家族史 | 动态血压、心脏超声、尿微量白蛋白 | 35岁起 |

---

## 四、生活方式风险加查项目

| 风险因素 | 建议加查项目 |
|---------|-------------|
| 长期吸烟 | 低剂量肺部CT（每年）、肺功能 |
| 长期饮酒 | GGT、肝脏超声、胃镜 |
| 久坐/肥胖 | HbA1c、血脂全套、颈动脉超声 |
| 高压职业 | 动态心电图、甲状腺功能 |
| 高盐饮食 | 尿微量白蛋白、心脏超声 |

---

## 五、复查时间建议

| 异常程度 | 复查建议 |
|---------|---------|
| 轻度异常，无症状 | 3-6个月后复查 |
| 中度异常 | 1-3个月后复查，建议专科就诊 |
| 严重异常 | 尽快就医，不建议等待复查 |
| 肿瘤标志物轻度升高 | 1个月后复查，观察动态变化 |
| 血脂/血糖临界 | 3个月后复查+生活方式干预 |
