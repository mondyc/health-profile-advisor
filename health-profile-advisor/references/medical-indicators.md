# 医学检验指标参考范围与临床意义

> 以下参考范围为成人通用参考值，不同医院/实验室可能略有差异，分析时以报告单上标注的参考范围为准。本文件供 Phase 2 异常标记时按需读取。

---

## 一、血常规

| 指标 | 参考范围 | 偏高常见原因 | 偏低常见原因 |
|------|----------|-------------|-------------|
| 白细胞 WBC | (4~10)×10⁹/L | 细菌感染、炎症、应激 | 病毒感染、免疫低下、药物影响 |
| 红细胞 RBC | 男(4.0~5.5)×10¹²/L；女(3.5~5.0)×10¹²/L | 脱水、缺氧、真性红细胞增多症 | 贫血、失血 |
| 血红蛋白 HGB | 男120~160g/L；女110~150g/L | 同RBC | 贫血（小细胞低色素→缺铁；大细胞→B12/叶酸缺乏） |
| 红细胞压积 HCT | 男0.40~0.50；女0.35~0.45 | 脱水 | 贫血 |
| 血小板 PLT | (100~300)×10⁹/L | 反应性增多、骨髓增殖性疾病 | ITP、药物影响、脾功能亢进 |
| 中性粒细胞% | 50%~70% | 细菌感染 | 病毒感染、化疗 |
| 淋巴细胞% | 20%~40% | 病毒感染、慢性炎症 | 免疫缺陷 |
| 单核细胞% | 3%~8% | 慢性感染、结核 | — |
| 嗜酸性粒细胞% | 0.5%~5% | 过敏、寄生虫 | — |

### 血常规解读要点
- 先看"三系"：白细胞系（感染/免疫）、红细胞系（贫血）、血小板系（凝血）
- 轻度异常未必有临床意义，需结合症状和其他指标综合判断
- 红细胞+血红蛋白+红细胞压积三者同步变化才有意义

## 二、肝功能

| 指标 | 参考范围 | 偏高常见原因 |
|------|----------|-------------|
| 谷丙转氨酶 ALT | 0~40 U/L | 肝细胞损伤（脂肪肝、药物、病毒性肝炎、饮酒） |
| 谷草转氨酶 AST | 0~40 U/L | 肝脏/心肌/骨骼肌损伤 |
| AST/ALT 比值 | <1 | >1 提示酒精性肝病或肝硬化可能 |
| 总胆红素 TBIL | 3.4~17.1 μmol/L | 肝细胞性黄疸、梗阻性黄疸、溶血 |
| 直接胆红素 DBIL | 0~6.8 μmol/L | 梗阻性黄疸、肝内胆汁淤积 |
| 间接胆红素 IBIL | 1.7~10.2 μmol/L | 溶血、Gilbert综合征 |
| 白蛋白 ALB | 35~55 g/L | ↓提示肝合成功能下降或营养不良 |
| 球蛋白 GLO | 20~35 g/L | ↑慢性炎症、自身免疫病、多发性骨髓瘤 |
| 白球比 A/G | 1.5~2.5 | ↓慢性肝病、自身免疫 |
| γ-谷氨酰转肽酶 GGT | 0~50 U/L | 酒精性肝病、胆道梗阻、药物性肝损伤 |
| 碱性磷酸酶 ALP | 40~150 U/L | 胆道梗阻、骨病、生长发育期可生理性升高 |

### 肝功能解读要点
- ALT 最敏感的肝细胞损伤标志；ALT > AST 多为非酒精性脂肪肝
- GGT 升高 + AST 升高 → 高度提示酒精性肝损伤
- 单项轻度异常（ALT 40~80）在体检中常见，多与脂肪肝、饮酒、药物有关
- 胆红素升高需区分直接/间接，判断黄疸类型

## 三、肾功能

| 指标 | 参考范围 | 异常意义 |
|------|----------|---------|
| 肌酐 Cr | 男53~106 μmol/L；女44~97 μmol/L | ↑肾小球滤过功能下降 |
| 尿素氮 BUN | 2.9~8.2 mmol/L | ↑肾脏排泄障碍、脱水、高蛋白饮食 |
| 尿酸 UA | 男149~416 μmol/L；女89~357 μmol/L | ↑痛风风险、代谢综合征相关 |
| eGFR | ≥90 ml/min | ↓肾功能减退（60~89轻度，30~59中度，<30重度） |
| 胱抑素C | 0.59~1.03 mg/L | ↑早期肾损伤标志，比肌酐更敏感 |

### 肾功能解读要点
- 肌酐升高往往意味着肾功能已损失50%以上，早期看 eGFR 和胱抑素C
- 尿酸升高与痛风相关，也是心血管和代谢综合征的独立危险因素
- 尿酸升高但无痛风发作 → 高尿酸血症，需生活方式干预

## 四、血脂

| 指标 | 参考范围 | 异常意义 |
|------|----------|---------|
| 总胆固醇 TC | <5.18 mmol/L | ↑动脉粥样硬化风险 |
| 甘油三酯 TG | <1.70 mmol/L | ↑心血管风险、急性胰腺炎风险（>5.6） |
| HDL-C（好胆固醇） | 男≥1.04；女≥1.30 mmol/L | ↓心血管保护不足 |
| LDL-C（坏胆固醇） | <3.37 mmol/L（无风险因素）；<2.59（高危） | ↑动脉粥样硬化核心危险因素 |
| 非HDL-C | <4.1 mmol/L | 优于LDL-C的综合风险评估 |
| 载脂蛋白A1 | 1.2~1.6 g/L | ↓类似HDL降低的风险 |
| 载脂蛋白B | 0.6~1.1 g/L | ↑类似LDL升高的风险 |
| Lp(a) | <300 mg/L | ↑遗传性心血管独立危险因素 |

### 血脂解读要点
- LDL-C 是心血管风险评估的首要指标
- TG 升高常与饮食（高糖高脂）、饮酒、糖尿病相关
- HDL-C 低比 TC 高更值得关注
- 血脂全套比单项检查价值大得多

## 五、血糖代谢

| 指标 | 参考范围 | 异常意义 |
|------|----------|---------|
| 空腹血糖 FPG | 3.9~6.1 mmol/L | 6.1~7.0 空腹血糖受损(IFG)；≥7.0 糖尿病 |
| 糖化血红蛋白 HbA1c | 4.0~6.0% | 5.7~6.4% 糖尿病前期；≥6.5% 糖尿病 |
| 餐后2h血糖 | <7.8 mmol/L | 7.8~11.1 糖耐量异常(IGT)；≥11.1 糖尿病 |
| 空腹胰岛素 | 2.6~24.9 μIU/mL | ↑胰岛素抵抗 |
| HOMA-IR | <2.5 | ↑胰岛素抵抗 |

### 血糖解读要点
- HbA1c 反映近 2-3 个月平均血糖，比单次空腹血糖更有价值
- 空腹血糖正常不代表没有问题，餐后血糖和 HbA1c 也需关注
- 糖尿病前期（IFG/IGT/HbA1c 5.7~6.4%）是逆转的关键窗口期

## 六、肿瘤标志物

| 指标 | 主要关联 | 注意事项 |
|------|---------|---------|
| AFP | 原发性肝癌 | 肝炎、肝硬化也可升高 |
| CEA | 结直肠癌、广谱 | 吸烟者可轻度升高 |
| CA199 | 胰腺癌、胆管癌 | 胆道炎症可升高 |
| CA125 | 卵巢癌 | 子宫内膜异位、盆腔炎可升高 |
| CA153 | 乳腺癌 | 良性乳腺疾病可升高 |
| PSA | 前列腺癌 | 前列腺增生、炎症可升高 |
| NSE | 小细胞肺癌 | 溶血标本可假性升高 |
| CYFRA21-1 | 非小细胞肺癌 | — |
| SCC | 鳞状细胞癌 | 皮肤病可升高 |
| ProGRP | 小细胞肺癌 | — |

### 肿瘤标志物解读要点（重要！）
- **轻度升高≠癌症**：炎症、良性病变、甚至生理状态都可能导致升高
- **必须注明免责**：每次涉及肿瘤标志物异常时，都必须提示"轻度升高原因多样，不一定代表恶性肿瘤，建议结合影像学检查综合评估"
- **动态监测更有价值**：单次轻度升高建议定期复查观察趋势，持续上升需警惕
- **组合筛查优于单项**：如肺癌组合 NSE+CYFRA21-1+ProGRP
